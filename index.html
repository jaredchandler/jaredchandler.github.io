Research questions I am interested in:

  • SQL Injection is the leading threat facing database systems. How can we use visualization to improve the security of relational databases?


  • Interfaces for writing SQL queries have remained relatively unchanged for the past 20 years. Can visualization help a user write a better query faster?


  • If I am allowed one pass over a large data structured file,what is the best thing I can produce and show to an analyst after that one pass?


  • Are the visualization community's theories of how visualizations should be used reflected in how lay users actually employ them? If not, why?


  • Do mental models of certain systems have a common visual representation across users?
